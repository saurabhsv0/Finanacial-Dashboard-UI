# Finanacial-Dashboard-UI
Built the dashboard in the Next app and repaired the malformed package manifest so the project now compiles cleanly. The screen now includes summary cards, a monthly balance trend, a category spending breakdown, searchable/filterable/sortable transactions, a frontend role switch, admin-only add/edit controls, insight cards, responsive layouts, empty states, and local persistence with `localStorage`.

Core implementation is in [page.js], styling is in [globals.css], the app shell is in [layout.js], and I fixed [package.json]. I used well-scoped React state rather than adding another state library.

#To run the file 
Open the project directory in the powershell, 
then run the command : 
"npm.cmd run dev" ,
Then open http://localhost:3000 
by pressing control + Enter/Left Click
